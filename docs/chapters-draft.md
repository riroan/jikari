## 멤버 매핑 자동화 전략

코드에서 카드 ID 일일이 매핑 안 함. import-chapters 스크립트가 다음 규칙으로 자동 매핑:

1. **vocab/sentence:** chapter 정의 파일에 `keywords: ["밥", "고기", ...]` 또는 `koreanMeanings: [...]` 매칭 시 자동 포함
2. **kanji:** chapter 정의 파일에 `kanji: ["東", "西", ...]` 명시 (소량이라 수기)
3. **grammar:** chapter 정의 파일에 `grammarIds: ["pattern-tai", "pattern-mashou", ...]` 명시
4. **particle (sentence category="particle"):** `blankParticles: ["は", "が"]` 매칭 시 자동 포함

scripts/chapters/chapters.config.ts (TypeScript) 한 파일에 위 정의 → import-chapters가 cards-store에서 매칭해서 chapter_members 테이블 생성.
